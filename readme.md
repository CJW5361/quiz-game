# 이미지 퀴즈 게임

## 프로젝트 소개
- 이미지를 보고 답을 맞추는 퀴즈 게임입니다.
- 총 20문제가 출제되며, 난이도별로 문제가 구성됩니다.
- 각 문제당 60초의 시간이 주어지며, 50초가 남았을 때 글자수가 제공됩니다.

## 게임 규칙
1. 정답을 맞추면 5점을 획득합니다.
2. 틀린 경우 같은 문제를 계속 풀 수 있습니다.
3. 시간 초과나 스킵시 다음 문제로 넘어갑니다.
4. 총 20문제를 모두 풀면 게임이 종료됩니다.

## 기술 스택
- Django
- SQLite
- HTML/CSS/JavaScript
